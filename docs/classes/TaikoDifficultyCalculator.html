Private _locallyCombinedDifficulty

  • _locallyCombinedDifficulty(colour: Colour, rhythm: Rhythm, staminaRight: Stamina, staminaLeft: Stamina, staminaPenalty: number): number
    • Defined in src/Difficulty/TaikoDifficultyCalculator.ts:191

    Returns the partial star rating of the beatmap, calculated using peak strains from all sections of the map. For each section, the peak strains of all separate skills are combined into a single peak strain for the section. The resulting partial rating of the beatmap is a weighted sum of the combined peaks (higher peaks are weighted more).

    Parameters

    • colour: Colour
    • rhythm: Rhythm
    • staminaRight: Stamina
    • staminaLeft: Stamina
    • staminaPenalty: number

    Returns number
